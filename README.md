# Omar AlzuBi — Portfolio

A personal portfolio built with React and Vite, showcasing projects, skills, and experience as a React Front-End Developer.

## Tech Stack

- React 18
- Vite
- React Router v6
- CSS Modules (BEM methodology)
- Web3Forms (contact form)

## Features

- Responsive design across all screen sizes
- Lazy-loaded pages with Suspense
- Cross-page hash navigation
- Animated SVG background icons
- Contact form with loading, success, and error states
- Accessible — focus-visible, ARIA labels, semantic HTML

## Getting Started

```bash
npm install
npm run dev
```

## Project Structure

```
src/
├── assets/         # Images, SVGs, data files
├── Components/     # Reusable UI components
├── Pages/          # Full page components
├── context/        # React context providers
├── hooks/          # Custom hooks
```
